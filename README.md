
 
 # Описание проекта
  
**Задача**: разработать DWH для для компании, занимающейся доставкой блюд из ресторанов. 
Пока есть только две системы-источника: 
 • MongoDB - система обработки заказов;  
 • PostgreSQL - система оплаты баллами.  

Этапы реализации:  
 • сбор и  анализ требований к DWН от заказчика; 
 • изучение исходных данных в источниках;
 • выбор модели данных для DWН и минимально необходимых слоев;  
 • разработка DWH. 

DWH реализовано на основе PostgreSQL и имеет следующие слои:    
 • STG слой для загрузки данных их источников as is; 
 • DDS слой детальных данных ("модель данных «снежинка»");   
 • CMD слой витрин.
