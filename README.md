# Описание проекта

**Задача**: разработать DWH для компании, занимающейся доставкой блюд из ресторанов.

## Системы-источники

Пока есть только две системы-источника:

- **MongoDB** — система обработки заказов
- **PostgreSQL** — система оплаты баллами

## Этапы реализации

1. Сбор и анализ требований к DWH от заказчика
2. Изучение исходных данных в источниках
3. Выбор модели данных для DWH и минимально необходимых слоев
4. Разработка DWH

## Архитектура DWH

DWH реализовано на основе **PostgreSQL** и имеет следующие слои:

- **STG слой** — для загрузки данных из источников *as is*
- **DDS слой** — детальных данных (модель данных «снежинка»)
- **CDM слой** — витрин
