Описание проекта  
Задача: разработать DWH  с использованием нескольких источников, реализовать витрины данных под бизнес-требования.  

Этапы реализации:  
 • сбор и  анализ требований к DWН от заказчика  
 • изучение изходных данных из источников (PostgreSQL, MongoDB)  
 • выбор модели данных для DWН и минимально необходимых слоев  
 • разработка DWH

в DWH будет 3 слоя:  
 • STG слой для загрузки данных их источников as is  
 • DDS слой детальных данных ("модель данных «снежинка»")   
 • CMD слой витрин  

Инструменты:  
• База данных: PostgreSQL, MongoDB  
• Оркестрация: Apache Airflow (набор DAG)
